配置 GitHub Token

因为每次保存都要写入 GitHub 仓库（生成一个 commit），需要一个 Personal Access Token。

  1. 打开 github.com/settings/tokens → Generate new token (Fine-grained)
  2. Repository access：选 Only select repositories → 勾 personal-dashboard
  3. Permissions → Repository permissions：把 Contents 设为 Read and write
  4. 生成后复制 Token（以 github_pat_ 开头），粘贴到下方
Token 仅保存在你这台浏览器的 localStorage 中，不会上传。
